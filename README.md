# To-Do List App 📝  

A simple to-do list app where users can add, remove, and mark tasks as completed.  

## Features  
✔️ Add new tasks  
✔️ Remove tasks  
✔️ Mark tasks as completed  

## Technologies Used  
- HTML  
- CSS  
- JavaScript  

## How to Run  
1. Download the project.  
2. Open `index.html` in a browser.
